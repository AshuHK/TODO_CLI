# TODO_CLI
A simple command line to-do list manager

- Keybindings:
  - Main Controls
    - l - list items in list
    - a - add new item to the list
    - c - complete/check off item in the list
    - d - delete item in the list

  - Misc.
    - h - help (print the keybindings)
    - q - quit TODO CLI
